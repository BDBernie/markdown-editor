#简易 markdown 编辑器

#[markdown-editor](http://qianjiahao.github.io/markdown-editor/)

基本功能

 - markdown [ 转义 ]
 - clean    [ 清空 ]
 - toHTML   [ 按html格式保存 ]
 - toText   [ 按txt格式保存 ]

使用less管理css样式，使用gulp管理项目

如需修改项目，请在修改后执行

	gulp
